Draw a function

This study is part of a research project conducted by Eric Schulz, University College London.

The study investigates how people learn and predict functions.

Please read the instructions carefully and try to perform as well as you can.

If you have any questions, you can contact e.schulz@ cs.ucl.ac.uk


Draw a function

Guidelines:


I. Below you see red dots that were drawn from an underlying function. The underlying function changes on every trial. It is your task to describe the underlying function as well as possible by drawing lines into the dots. There will be 50 trials in total.

II. Dots will never be exactly at the position of the true underlying function.

III. There will also be different numbers of dots from trial to trial; sometimes there will be more, sometimes less.

IV. Lines are drawn by placing dots on the screen which then will be automatically connected. You can place as many dots as you want. The first dot has to be placed at the leftmost side, the last dot at the rightmost side.

V. The two blue lines mark an area without any dots. It is your task to draw the function for this area just as you would do for any other area.
